\section{实验目的}

\begin{enumerate}
    \item 进行电路设计，加深对差分放大器、电流源、射随器、负反馈及稳定性等理论知识的理解；
    \item 加深对运放原理、参数的理解；
    \item 理解正弦波振荡器振荡条件，加深对负反馈放大器稳定性的理解；
    \item 理解文氏桥 (Wien-Bridge) 的选频特性，利用设计的运放搭建文氏振荡器。
\end{enumerate}
